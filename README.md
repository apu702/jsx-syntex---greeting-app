# jsx-syntex---greeting-app
Dynamic Greeting Page (React JSX Project)
Overview:

A simple React app that displays a dynamic greeting based on the user’s current time: “Good morning”, “Good afternoon”, or “Good evening” and change the Emoji and show the time after 1 hour. Also here is some basic styles.

Learning Outcomes

JSX syntax:

Embedding JavaScript expressions in JSX

Conditional rendering using if/else

How It Works:

The app gets the current hour from the system and displays a greeting depending on whether it’s morning, afternoon, or evening.

Conclusion:

A beginner-friendly project demonstrating dynamic content in React using basic JSX and conditional logic.# jsx-syntex---greeting-app
